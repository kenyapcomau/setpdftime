# setpdftime
```
usage: setpdftime.py [-h] [-v] [-n] file [file ...]

Set PDF file modification time to ModDate value in metadata

positional arguments:
  file           PDF file

optional arguments:
  -h, --help     show this help message and exit
  -v, --verbose  show equivalent touch command
  -n, --no-run   don't actually change the modification time
```
